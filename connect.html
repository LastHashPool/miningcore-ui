Miner Configuration

This is the basic guide how to setup your miner to this pool.

Mine with

NVIDIA GPUs using ccminer

We recommend using the z-enemy fork of ccminer, because it gives best results for hashing speed and stability. You can download your desired miner from the table below, after that start the miner with the following command:

ccminer.exe -a x16r -o stratum+tcp://evocation.network:5000 -u you_raven_coin_address -p x

Fork Download
z-enemy(1.08) - closed source, 1% dev fee mega.nz
nevermore - open source, 1% dev fee github
suprminer - open source, no dev fee github

Mine with

AMD GPUs using sgminer

We recommend using the avermore fork, because it gives best results for hashing speed and stability. You can download your desired miner from the table below, after that start the miner with the following command:

sgminer -k x16r -o stratum+tcp://evocation.network:5000 -u you_raven_coin_address -p x

Fork Download
avermore - open source, 1% dev fee github
